# 🎮 Video Game Market Insights: Identifying Success Patterns

This project focuses on a comprehensive **Exploratory Data Analysis (EDA)** of the global video game market. Working with historical data up to 2016, the objective was to identify the key factors that determine a game's commercial success to plan data-driven advertising campaigns for the online store **Ice**.

## 🎯 Project Goal
Analyze sales, user/expert reviews, and platform trends to forecast which games and platforms are likely to be profitable in the upcoming year.

## 🛠️ Tech Stack
- **Languages:** Python (Pandas, NumPy)
- **Data Visualization:** Matplotlib, Seaborn
- **Statistical Analysis:** Hypothesis Testing (T-tests for user ratings)

## 📈 Key Findings
- **Platform Lifecycle:** Identified the "boom and bust" cycle of consoles (PS4, Xbox One, 3DS), noting that platforms typically have a 10-year relevance window.
- **Regional Preferences:** Significant divergence found in the Japanese market (preference for handhelds like 3DS) compared to North America and Europe (preference for home consoles).
- **Critic vs. User Impact:** Statistical analysis showed that professional critic scores have a stronger correlation with global sales than user scores.
- **Top Genres:** Action and Shooter genres lead in total sales, while Platformers show higher average sales per game.

## 📁 Repository Structure
- `Game_Market_Insights_2016.ipynb`: Complete data cleaning, visualization, and statistical testing.
- `requirements.txt`: Necessary Python libraries to run the analysis.
